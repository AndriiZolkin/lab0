1.Что понимается под термином «.NET Framework»?
- «.NET Framework» - программная платформа, основой платформы является общеязыковая среда исполнения (CLR), которая подходит для различных языков программирования.
NET Framework определяет среду для поддержки создания и выполнения платформонезависимых гетерогенных приложений.
